## Usage

Insert this html tag to your page.

```html
<img src="https://yourdomain.example.com/counter.svg">
```

That's all.

## API

### ``/counter.svg``

Returns counter image.

#### Query

- `offset` (number, default: 0)
  - Offset of counts.
- `color` (none or any, default: none)
  - Color-Cats mode.
